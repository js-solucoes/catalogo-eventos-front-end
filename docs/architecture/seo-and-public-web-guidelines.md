# SEO and public web guidelines

## Quando SEO é requisito crítico

- páginas públicas de descoberta orgânica
- landing pages
- páginas institucionais
- páginas de marca
- conteúdo indexável voltado a aquisição

## Diretrizes

- considerar renderização adequada para indexação
- definir estratégia de metadata por página
- garantir semântica HTML correta
- otimizar performance web
- considerar sitemap, robots e canonical quando necessário
- instrumentar tag manager com governança mínima
- evitar excesso de scripts que prejudiquem performance e indexação

## Avaliação arquitetural

Quando SEO for prioritário, avaliar se SPA pura é suficiente.
A decisão deve considerar:

- volume de conteúdo público dinâmico
- dependência de JavaScript para conteúdo essencial
- necessidade de indexação robusta
- simplicidade operacional
- custo e manutenibilidade
