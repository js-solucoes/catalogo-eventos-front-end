# Frontend observability guidelines

## Objetivo

Garantir mensuração, monitoramento mínimo e boa governança de scripts e instrumentação.

## Diretrizes

- definir eventos relevantes de navegação e negócio
- separar mensuração entre área pública e CRM quando fizer sentido
- evitar scripts soltos espalhados por componentes
- preferir pontos de integração claros para analytics
- monitorar erros críticos do front quando aplicável
- considerar impacto de scripts em performance
